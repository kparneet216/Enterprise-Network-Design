# Network Planning

## Project Objective

The objective of this project was to design and implement a small enterprise network for a fictional company called TechSolutions Pty Ltd using Cisco Packet Tracer.

The network demonstrates enterprise networking concepts including VLAN segmentation, inter-VLAN routing, DHCP, NAT, and connectivity verification.

---

## Network Requirements

The company consists of three departments:

- Human Resources
- Finance
- IT

Each department requires its own VLAN for improved security and network organization.

---

## Devices Used

- Cisco 2911 Router
- Cisco 2960 Switch
- 3 PCs
- Packet Tracer Cloud

---

## Technologies Used

- VLANs
- Router-on-a-Stick
- DHCP
- NAT/PAT
- TCP/IP
